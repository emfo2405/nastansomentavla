# WordPress projekt

Syftet med det här projektet var att skapa en webbplats i form av en företagspresentation. I mitt fall skapade jag en webbplats för företaget Nästan som en tavla som agerar som en företagspresentation och där jag även har utvecklat ett eget tema. Temat baserades på en egenskapad HTML och CSS-grund. I temat finns det mallar för standardsidor men även för sidor där kategorier specificeras, så som nyheter eller FAQ. På webbplatsen är även webbshopsfunktionalitet implementerad med WooCommerce och därmed har även temafiler för produkter, varukorg och kassa skapats.
